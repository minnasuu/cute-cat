# 2026-03-11

## 邮件发送 Markdown 内联样式优化

`email-send.ts` 的 `mdToHtml` 函数增加了 `injectEmailStyles` 后处理步骤，为 Markdown 转换后的 HTML 标签（h1-h3、p、ul/ol/li、blockquote、code/pre、a、table/th/td、hr）注入邮件客户端兼容的内联样式，确保邮件正文中的 Markdown 内容（标题、列表、代码块、表格等）在各邮件客户端中正确美观地渲染。

## 工作流画布 Dify 风格优化

参考 Dify 画布交互体验，对 `workflow-canvas/` 进行了全面升级：

### 新增文件
- **`useEdgeDrag.ts`** - 拖拽连线 hook，状态机管理（端口检测 via `elementFromPoint`、画布坐标转换、连接触发）
- **`useCanvasKeyboard.ts`** - 快捷键 hook（Delete 删除节点/连线、Escape 取消选中、Space 画布拖拽模式）
- **`DragPreviewLine.tsx`** - 拖拽连线预览组件（虚线贝塞尔曲线+发光效果+鼠标圆点）
- **`Minimap.tsx`** - Canvas 2D 小地图组件（缩略节点/连线、视口范围框、点击/拖拽快速定位）

### 改造文件
- **`canvas-utils.ts`** - 新增 `PORT_SIZE`、`getInputPortPos/getOutputPortPos/getStartOutputPortPos` 端口坐标计算、`canConnect` 连接校验
- **`StepNode.tsx`** - 添加 input/output 端口圆点 UI（hover 放大高亮）、端口 pointerdown 触发连线拖拽、isDropTarget 高亮
- **`EdgeLines.tsx`** - 连线 hover 高亮+发光效果、选中态流动动画（stroke-dasharray）、中点删除按钮
- **`WorkflowCanvas.tsx`** - 集成 useEdgeDrag/useCanvasKeyboard/DragPreviewLine，开始节点增加输出端口
- **`WorkflowEditorPage.tsx`** - 新增 handleConnect/handleDeleteEdge 回调、集成 Minimap（含 ResizeObserver 跟踪容器尺寸）
